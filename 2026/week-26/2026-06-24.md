## Active Threats

- **[MISP Core Broken Access Control — CVE-2026-56423 & CVE-2026-56424 (CVSS 8.8)](https://nvd.nist.gov/vuln/detail/CVE-2026-56423)** — Two HIGH-severity broken access control flaws in MISP Core allow authenticated lower-privileged users to delete Event Reports and Sharing Groups belonging to other organizations, and to perform unauthorized writes across multiple subsystems where ownership checks were missing. Both vulnerabilities affect bulk deletion flows where authorization was validated against role-level permissions rather than per-object ownership. Patch MISP immediately. *Source: NVD*

- **[CVE-2026-35018: NetComm NF20MESH Router Authenticated RCE via Shell Injection (CVSS 8.8)](https://nvd.nist.gov/vuln/detail/CVE-2026-35018)** — NetComm NF20MESH routers running firmware R6B031 and earlier allow authenticated attackers to execute arbitrary commands as root by injecting shell metacharacters into the username JSON parameter processed by dalStorage_addUserAccount. No patch version was specified in the disclosure; users should monitor NetComm's advisory channel. *Source: NVD*

- **[Cisco Catalyst SD-WAN CVE-2026-20262 Added to CISA KEV — Federal Patch Deadline June 29](https://www.securityweek.com/cisco-patches-another-sd-wan-zero-day-the-sixth-exploited-in-2026/)** — CISA added CVE-2026-20262 — an arbitrary file-write vulnerability in Cisco Catalyst SD-WAN Manager exploitable by authenticated remote attackers to escalate privileges to root via a crafted API request — to the Known Exploited Vulnerabilities catalog on June 22.
