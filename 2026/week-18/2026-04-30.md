- **[CVE-2026-7418 and CVE-2026-7419 (CVSS 8.8): Remotely Exploitable Buffer Overflow in UTT HiPER 1250GW Routers](https://nvd.nist.gov/vuln/detail/CVE-2026-7418)** — Two remotely exploitable stack-based buffer overflow vulnerabilities in UTT HiPER 1250GW routers (firmware <=3.2.7-210907-180535) allow attackers to trigger arbitrary code execution via crafted Profile parameters in the NTP and formTaskEdit_ap functions; public exploits are available. *Source: NVD*
